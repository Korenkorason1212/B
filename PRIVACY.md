Yes, we maintain a clear policy regarding user data privacy and usage within our community:

Data Collection & Usage:
Our application only processes user IDs, message content, and server role data in real-time to execute requested administrative slash commands (/strike, /promotion, /trigger) and to scan for preset support keywords. No user data, message history, or personal identification information is sold, shared with third parties, or used for tracking.

Data Storage & Security:
Custom word triggers set by server managers are saved locally in a secure triggers.json configuration file on our private hosting environment. Disciplinary logs and promotion tracking data are transmitted directly to designated secure logging channels inside our Discord server. No data is stored externally on unsecured public databases.

User Rights & Data Removal:
Users can request the complete removal of any saved server configurations or logs at any time by contacting the server administration team directly.
