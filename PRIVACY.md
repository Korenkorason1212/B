# Privacy Policy for THM Bot

**Last Updated: June 23, 2026**

This Privacy Policy explains how THM Bot ("the bot") processes and handles data within Discord servers. We value your privacy and aim to be completely transparent.

## 1. Data We Access & Process
To function correctly, THM Bot utilizes specific Discord Privileged Intents to process the following data in real-time:
* **Message Content:** The bot processes the text content of messages solely to detect predefined, server-specific keyword triggers. **Messages are analyzed instantly in volatile memory and are never stored, logged, or saved to any database.**
* **Server Member Data & Role IDs:** The bot accesses server member lists and roles strictly to execute administrative commands like `/promotion` and `/strike`. This data is used only to modify user roles within the server hierarchy as directed by server administrators.

## 2. Data Storage & Retention
* **Administrative Logs:** Data regarding server infractions (strikes) or promotional history is stored securely. This data consists only of user IDs and the infraction reason, and is accessible exclusively by authorized server administrators.
* **No Third-Party Sharing:** We do not sell, trade, rent, or share any user or server data with third-party services or individuals.

## 3. Data Deletion & Contact
If you are a server owner or user and wish to have your server's administrative logs or data removed from the bot, please contact the bot developer directly through our official support channels or via Discord.
